# BaoGuaWindowsEvent  
## 说明  
宝瓜Windows日志分析器，一款简洁方便的Windows日志分析工具。  
使用C#开发，需要依赖.net core 8.0框架，下载地址：https://aka.ms/dotnet-core-applaunch?missing_runtime=true&arch=x64&rid=win-x64&os=win10&apphost_version=8.0.2&gui=true
